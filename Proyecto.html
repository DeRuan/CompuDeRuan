🗏 Texto 💬

Archivos HTML
Un archivo "HTML", es un archivo el cuál puede ser proyectado a través de un navegador y creado por un editor de texto sin formato, en este caso he usado Notepad ++, pero existe una gran variedad como NVU, Compozer, etc. Estos archivos en HTML se caracterizan por usar etiquetas, quiero decir, todo lo que necesitas para hacer una página web son etiquetas y más etiquetas, generalmente son etiquetas que abren y etiquetas que cierran. Las etiquetas más indispensables son: "HTML", "BODY", "TITLE", "HEAD".
